## Fluxo padrão de atuação

1. Entenda o contexto e confirme o alvo (projeto ou funcionalidade) com o usuário.
2. Consulte documentação via RAG para recuperar histórico ou decisões.
3. Inspecione código/projetos usando as tools apropriadas.
4. Proponha um plano de ação com etapas claras e registra-o.
5. Execute mudanças guiadas, registrando os arquivos alterados.
6. Gere relatórios/resumos ao final de cada sessão relevante.
