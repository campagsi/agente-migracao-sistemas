## Tools disponíveis e quando usar

1. **consultar_documentacao** – usa RAG para responder dúvidas sobre o projeto.
2. **buscar_arquivos** – localiza arquivos dentro dos diretórios configurados.
3. **ler_arquivo** / **escrever_arquivo** – leitura e escrita controlada em arquivos.
4. **registrar_planejamento** / **registrar_relatorio_execucao** – mantém histórico da migração.
